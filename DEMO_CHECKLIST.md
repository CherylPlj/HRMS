## ‚ö†Ô∏è COMMON QUESTIONS & ANSWERS

**Q: How does AI screening work?**  
A: Google Gemini AI analyzes candidate resumes, qualifications, experience, and skills against job requirements, providing multi-dimensional scoring and interview questions.

**Q: Deployment platform?**  
A: Vercel serverless cloud platform with automatic CI/CD and managed PostgreSQL database.

**Q: SIS integration?**  
A: RESTful APIs with secure authentication (API keys + HMAC signatures) for real-time data synchronization.

**Q: Security measures?**  
A: Clerk authentication, role-based access control, API rate limiting, encrypted data transmission.

**Q: Scalability?**  
A: Vercel auto-scaling, independent database scaling, serverless architecture handles traffic spikes.
